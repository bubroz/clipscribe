# Security Policy

Last Updated: August 11, 2025

## Supported Versions
Private repository; security updates are applied to `main` on a rolling basis.

## Reporting a Vulnerability
- Email: zforristall@gmail.com
- Please include: description, reproduction steps, impact, and logs.
- We will acknowledge within 5 business days and provide a remediation timeline when applicable.

## Disclosure
- Do not publicly disclose vulnerabilities until a fix has been validated or we have agreed otherwise.

## Scope
- This repository and deployed API services.

Thank you for helping keep ClipScribe secure.
