# Contributing

This is a private, proprietary project. Contributions are by invitation only.

## Guidelines for Invited Contributors
- Discuss scope and design before opening a PR
- Add tests for new or changed behavior
- Update relevant documentation (README, docs/, CHANGELOG)
- Follow formatting (Black), type hints, and existing patterns

## Commit Messages
- Conventional commits preferred (feat:, fix:, docs:, chore:, test:)

## PR Review
- At least one code owner approval required

Thank you for contributing to ClipScribe.
