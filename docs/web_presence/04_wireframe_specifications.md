# Wireframe Specifications
**ClipScribe v3.0.0 Web Presence**  
**Date:** November 13, 2025  
**Purpose:** Detailed wireframe specifications for UI/UX designer to implement in Figma

---

## Overview

This document contains pixel-perfect wireframe specifications for all ClipScribe pages. Each page includes:
1. Desktop layout (1440px width)
2. Mobile layout (375px width)
3. Content block dimensions
4. Interaction specifications
5. Responsive behavior notes

**For UI/UX Designer:**
- Use these specs to create high-fidelity designs in Figma
- All content blocks are defined with exact hierarchy
- Annotations explain interactions and behaviors
- Mobile layouts show stacking/reflow patterns

---

## Design System Foundation

### Typography Hierarchy

**Headings:**
- H1: 48-64px (Hero headlines)
- H2: 36-48px (Section headlines)
- H3: 24-32px (Subsection headlines)
- H4: 18-24px (Card titles)

**Body:**
- Large: 18-20px (Subheadlines, important body)
- Regular: 16px (Standard body text)
- Small: 14px (Captions, metadata)

### Spacing System

**Use 8px grid:**
- Micro: 8px
- Small: 16px
- Medium: 24px
- Large: 32px
- XLarge: 48px
- XXLarge: 64px
- Massive: 96px

### Component Patterns

**Buttons:**
- Primary: Filled, high contrast
- Secondary: Outlined
- Text: No background

**Cards:**
- White/light background
- Subtle shadow or border
- 8-16px padding
- 8px border radius

**Tables:**
- Header row with subtle background
- Alternating row colors (optional)
- Clear borders
- Hover state on rows

## Accessibility Specifications

### Color Contrast

**Text:**
- Body text: 4.5:1 minimum
- Large text (18px+): 3:1 minimum
- Headings: 4.5:1 minimum

**Interactive Elements:**
- Buttons: 3:1 contrast with background
- Links: Underlined or 3:1 contrast
- Focus indicators: 3:1 contrast

### Keyboard Navigation

**All interactive elements:**
- Focusable via Tab key
- Clear focus indicators
- Skip to content link
- Escape key closes modals/menus

### Screen Reader Support

**Images:**
- Alt text for all images
- Decorative images: alt=""

**Semantic HTML:**
- Proper heading hierarchy
- nav, main, footer elements
- aria-labels where needed

## Animation Guidelines

### Subtle, Fast, Purposeful

**Allow:**
- Button hover states (100-200ms)
- Card lifts on hover (200ms)
- Smooth scrolls (500ms)
- Accordion expand/collapse (300ms)
- Fade-ins on scroll (optional, 300ms)

**Avoid:**
- Page load animations
- Auto-playing videos
- Parallax effects
- Excessive motion

**Respect:**
- prefers-reduced-motion media query
- Disable all animations if user prefers

## Performance Requirements

### Load Time

**First Contentful Paint:** <1.5 seconds  
**Largest Contentful Paint:** <2.5 seconds  
**Time to Interactive:** <3.5 seconds

### Optimizations

**Images:**
- WebP with JPEG fallback
- Lazy loading below fold
- Proper sizing (no oversized images)

**Fonts:**
- System fonts (no web fonts) OR
- 1-2 web fonts max, preloaded

**Code:**
- Minified CSS/JS
- Critical CSS inline
- Defer non-critical JS
