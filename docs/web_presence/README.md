## Key Decisions Made

### Business Model: Hybrid Open Source + Paid Service
- Free tier: Self-hosted MIT licensed CLI (unlimited)
- Paid tier: Managed service ($25-500 per order)
- Model proven by Supabase, PostHog, Cal.com

### Pricing Structure: Generous Launch Pricing
- Free: Unlimited (open source)
- Single Video: $25-50 (one video, all formats)
- Series: $200-500 for 5-20 videos (not 3-10) - GENEROUS
- Enterprise: Custom (white-glove, API access)

### Tone: Direct, Honest, Technical
- No marketing fluff or buzzwords
- Show exact costs and margins
- Transparent about business model
- Confident but not arrogant

### Design Vibe: Clean, Minimal, Professional
- Inspired by PostHog, Supabase, Linear
- Fast loading (<3 seconds)
- Mobile-first responsive
- Accessible (WCAG 2.1 AA)

### MVP Approach: Simple First, Automate Later
- Phase 1: Static site + manual processing (Dec 2025)
- Phase 2: Automated upload + processing (Jan-Feb 2026)
- Phase 3: Enterprise features + API (Mar+ 2026)

## Success Criteria

**Design is successful if it:**

1. **Communicates clearly** - Visitor understands what ClipScribe is in 5 seconds
2. **Guides action** - Clear paths to GitHub (free) or email (paid)
3. **Builds trust** - Samples and transparency make it believable
4. **Works everywhere** - Mobile, desktop, all browsers
5. **Loads fast** - <3 seconds on broadband, <5 seconds on 3G
6. **Looks professional** - Real company, not hobby project
7. **Converts** - >20% of visitors take action (GitHub or email)
